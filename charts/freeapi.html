Free API

Free to use, very basic API is already avaliable. New functionalities will be added soon to address ASA managers data needs.

Usage: get rate for a pair of assets using Tinyman URL structure.

Try it out. Example:

https://freetinycharts.ovh/api/?asset_in=163650&asset_out=31566704

Returns latest price saved for asset:

{"status":200,"status_message":"Found","data":"0.11800554578862989"}
